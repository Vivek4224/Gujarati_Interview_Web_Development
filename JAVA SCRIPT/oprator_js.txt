Here are 50 JavaScript operator-related interview questions and answers:

### 1. **What are operators in JavaScript?**
   **Answer**: Operators are symbols or keywords used to perform operations on values and variables. They can be categorized into arithmetic, assignment, comparison, logical, and other types.

### 2. **What are arithmetic operators in JavaScript?**
   **Answer**: Arithmetic operators are used to perform mathematical calculations on numbers. They include:
   - `+` (Addition)
   - `-` (Subtraction)
   - `*` (Multiplication)
   - `/` (Division)
   - `%` (Modulus)
   - `++` (Increment)
   - `--` (Decrement)

### 3. **What is the difference between `++i` and `i++` in JavaScript?**
   **Answer**: 
   - `++i` is the **pre-increment** operator. It increments the value of `i` before it is used.
   - `i++` is the **post-increment** operator. It uses the value of `i` first, and then increments it.

### 4. **What are assignment operators in JavaScript?**
   **Answer**: Assignment operators are used to assign values to variables. They include:
   - `=` (Assign)
   - `+=` (Add and assign)
   - `-=` (Subtract and assign)
   - `*=` (Multiply and assign)
   - `/=` (Divide and assign)
   - `%=` (Modulus and assign)

### 5. **What are comparison operators in JavaScript?**
   **Answer**: Comparison operators are used to compare two values. They include:
   - `==` (Equal to)
   - `===` (Strict equal to)
   - `!=` (Not equal to)
   - `!==` (Strict not equal to)
   - `>` (Greater than)
   - `<` (Less than)
   - `>=` (Greater than or equal to)
   - `<=` (Less than or equal to)

### 6. **What is the difference between `==` and `===`?**
   **Answer**:
   - `==` compares values for equality with type coercion.
   - `===` compares both values and types for equality (strict equality).

### 7. **What are logical operators in JavaScript?**
   **Answer**: Logical operators are used to perform logical operations, typically on boolean values:
   - `&&` (Logical AND)
   - `||` (Logical OR)
   - `!` (Logical NOT)

### 8. **What is the ternary operator in JavaScript?**
   **Answer**: The ternary operator is a shorthand for an `if-else` statement. Syntax:
   ```javascript
   condition ? expression_if_true : expression_if_false;
   ```

### 9. **What are bitwise operators in JavaScript?**
   **Answer**: Bitwise operators perform operations on the binary representations of numbers. They include:
   - `&` (AND)
   - `|` (OR)
   - `^` (XOR)
   - `~` (NOT)
   - `<<` (Left shift)
   - `>>` (Right shift)
   - `>>>` (Unsigned right shift)

### 10. **What is the purpose of the `typeof` operator in JavaScript?**
   **Answer**: The `typeof` operator is used to determine the type of a given operand (variable or expression). Example:
   ```javascript
   typeof "hello"; // "string"
   typeof 123; // "number"
   ```

### 11. **What is the purpose of the `instanceof` operator in JavaScript?**
   **Answer**: The `instanceof` operator is used to check if an object is an instance of a specific class or constructor function. Example:
   ```javascript
   obj instanceof Array; // Returns true if obj is an Array
   ```

### 12. **What are the assignment operators in JavaScript?**
   **Answer**: Assignment operators are used to assign values to variables. Examples include:
   - `=` (Assign value)
   - `+=` (Add and assign)
   - `-=` (Subtract and assign)
   - `*=` (Multiply and assign)
   - `/=` (Divide and assign)
   - `%=` (Modulus and assign)

### 13. **What is the `in` operator in JavaScript?**
   **Answer**: The `in` operator is used to check if a property exists in an object. Example:
   ```javascript
   "name" in obj; // Returns true if "name" is a property of obj
   ```

### 14. **What is the `delete` operator in JavaScript?**
   **Answer**: The `delete` operator is used to remove a property from an object or an element from an array. Example:
   ```javascript
   delete obj.name; // Deletes the "name" property from obj
   ```

### 15. **What is the `void` operator in JavaScript?**
   **Answer**: The `void` operator is used to evaluate an expression and return `undefined`. It is often used to ensure a function doesn't return a value. Example:
   ```javascript
   void(0); // Returns undefined
   ```

### 16. **What is the `typeof` operator used for?**
   **Answer**: The `typeof` operator is used to return the data type of a variable or expression. Example:
   ```javascript
   typeof 42; // "number"
   typeof "hello"; // "string"
   ```

### 17. **What is the `??` (nullish coalescing) operator?**
   **Answer**: The `??` operator returns the right-hand operand if the left-hand operand is `null` or `undefined`, otherwise it returns the left-hand operand. Example:
   ```javascript
   let x = null ?? "default"; // "default"
   ```

### 18. **What is the `&&` (AND) operator used for?**
   **Answer**: The `&&` operator returns `true` if both operands are true. It is often used in conditional checks:
   ```javascript
   if (x && y) { // Executes if both x and y are truthy
   }
   ```

### 19. **What is the `||` (OR) operator used for?**
   **Answer**: The `||` operator returns `true` if at least one operand is true. It is commonly used in default value assignments:
   ```javascript
   let result = x || y; // Returns x if x is truthy, otherwise returns y
   ```

### 20. **What is the `!` (NOT) operator used for?**
   **Answer**: The `!` operator negates the boolean value of its operand. Example:
   ```javascript
   !true; // false
   !false; // true
   ```

### 21. **What is the `<<` (left shift) operator in JavaScript?**
   **Answer**: The left shift (`<<`) operator shifts the bits of a number to the left, filling with zeros on the right. Each shift operation is equivalent to multiplying by `2`. Example:
   ```javascript
   5 << 1; // 10 (binary shift of 101 to 1010)
   ```

### 22. **What is the `>>` (right shift) operator in JavaScript?**
   **Answer**: The right shift (`>>`) operator shifts the bits of a number to the right. Each shift operation is equivalent to dividing by `2`. Example:
   ```javascript
   5 >> 1; // 2 (binary shift of 101 to 10)
   ```

### 23. **What is the `>>>` (unsigned right shift) operator?**
   **Answer**: The unsigned right shift (`>>>`) operator shifts the bits of a number to the right and fills with zeros, even for negative numbers. Example:
   ```javascript
   -5 >>> 1; // 2147483642
   ```

### 24. **What is the difference between `==` and `===`?**
   **Answer**: 
   - `==` performs equality comparison with type coercion.
   - `===` performs strict equality comparison without type coercion.

### 25. **What is the `+=` operator used for?**
   **Answer**: The `+=` operator adds the right-hand operand to the left-hand operand and assigns the result to the left-hand operand. Example:
   ```javascript
   x += 5; // x = x + 5
   ```

### 26. **What is the `-=` operator used for?**
   **Answer**: The `-=` operator subtracts the right-hand operand from the left-hand operand and assigns the result to the left-hand operand. Example:
   ```javascript
   x -= 5; // x = x - 5
   ```

### 27. **What is the `*=` operator used for?**
   **Answer**: The `*=` operator multiplies the left-hand operand by the right-hand operand and assigns the result to the left-hand operand. Example:
   ```javascript
   x *= 5; // x = x * 5
   ```

### 28. **What is the `/=` operator used for?**
   **Answer**: The `/=` operator divides the left-hand operand by the right-hand operand and assigns the result to the left-hand operand. Example:
   ```javascript
   x /= 5; // x = x / 5
   ```

### 29. **What is the `%=` operator used for?**
   **Answer**: The `%=` operator computes the remainder of dividing the left-hand operand by the right-hand operand and assigns the result to the left-hand operand. Example:
   ```javascript
   x %= 5; // x = x % 5
   ```

   condition ? expr1 : expr2;

### 36. **What is the `??` (nullish coalescing) operator in JavaScript?**
   **Answer**: The `??` operator returns the right-hand operand if the left-hand operand is `null` or `undefined`. Example:
   ```javascript
   let result = value ?? "default";
   ```

### 37. **What is the `&&` (logical AND) operator used for?**
   **Answer**: The `&&` operator checks if both operands are truthy. If they are, it returns the second operand; otherwise, it returns the first falsy operand. Example:
   ```javascript
   x && y; // Returns y if both x and y are truthy
   ```

### 38. **What is the `||` (logical OR) operator used for?**
   **Answer**: The `||` operator checks if either operand is truthy. If one is truthy, it returns the first truthy operand; otherwise, it returns the second operand. Example:
   ```javascript
   x || y; // Returns x if x is truthy, otherwise returns y
   ```

### 39. **What is the `!` (logical NOT) operator used for?**
   **Answer**: The `!` operator negates the truthiness of its operand. Example:
   ```javascript
   !true; // false
   ```

### 40. **What is the `&` (bitwise AND) operator used for?**
   **Answer**: The `&` operator performs a bitwise AND on two numbers. It compares each bit of the numbers and returns 1 if both bits are 1. Example:
   ```javascript
   5 & 3; // 1
   ```

### 41. **What is the `|` (bitwise OR) operator used for?**
   **Answer**: The `|` operator performs a bitwise OR on two numbers. It returns 1 if at least one of the bits is 1. Example:
   ```javascript
   5 | 3; // 7
   ```

### 42. **What is the `^` (bitwise XOR) operator used for?**
   **Answer**: The `^` operator performs a bitwise XOR (exclusive OR) on two numbers. It returns 1 only if one of the bits is 1, but not both. Example:
   ```javascript
   5 ^ 3; // 6
   ```

### 43. **What is the `~` (bitwise NOT) operator used for?**
   **Answer**: The `~` operator inverts all the bits of its operand. Example:
   ```javascript
   ~5; // -6
   ```

### 44. **What is the `<<` (left shift) operator used for?**
   **Answer**: The `<<` operator shifts the bits of a number to the left. Example:
   ```javascript
   5 << 1; // 10
   ```

### 45. **What is the `>>` (right shift) operator used for?**
   **Answer**: The `>>` operator shifts the bits of a number to the right. Example:
   ```javascript
   5 >> 1; // 2
   ```

### 46. **What is the `>>>` (unsigned right shift) operator used for?**
   **Answer**: The `>>>` operator shifts bits to the right and fills the leftmost bits with zeros. Example:
   ```javascript
   -5 >>> 1; // 2147483642
   ```

### 47. **What is the `+=` operator used for in strings?**
   **Answer**: The `+=` operator concatenates two strings and assigns the result to the left operand. Example:
   ```javascript
   let str = "Hello";
   str += " World"; // "Hello World"
   ```

### 48. **What is the `==` operator used for?**
   **Answer**: The `==` operator compares two values for equality, performing type coercion if necessary. Example:
   ```javascript
   5 == "5"; // true
   ```

### 49. **What is the `===` operator used for?**
   **Answer**: The `===` operator compares two values for strict equality (value and type). Example:
   ```javascript
   5 === "5"; // false
   ```

### 50. **What is the `!=` operator used for?**
   **Answer**: The `!=` operator compares two values for inequality, with type coercion. Example:
   ```javascript
   5 != "5"; // false
   ```
